frontend-nanodegree-arcade-game
===============================
# Game Features:
1. The player can be moved to the right, left, up and down using the keypad's left, right, up and down keys.
2. If the player reaches to the end the game will reset and player will move back to the original position.
3. When player collides with the bug the game will reset and the player is palced in the original position.
4. The bugs move in random loop and speed.
